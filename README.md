# slider-project-js

## Description
This is a simple slider project, created with HTML, CSS and JavaScript.
It allows user to slide between items by clicking on arrow up and down, and by pressing
key Uo and Down.

## Technologies
- HTML
- CSS 
- Vanila JavaScript
